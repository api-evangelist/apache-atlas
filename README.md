# Apache Atlas (apache-atlas)
Apache Atlas is a scalable and extensible set of core foundational governance services that enables enterprises to effectively meet their compliance requirements within Hadoop and allows integration with the whole enterprise data ecosystem. It provides metadata management, data classification, and lineage tracking.
